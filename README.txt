Cara menjalankan aplikasi
- pertama jalankan dulu xamppnya
- belum ada?, download dulu disini -> https://www.apachefriends.org/download.html
- instal xampp yang sudah di download tadi
- setelah berhasil diinstal, buat folder baru di "C:/xampp/htdocs" (Jika xampp di instal di drive C), namakan folder dengan "app1"
- ekstak di folder yang barusan dibuat ("C:/xampp/htdocs/app1")
- jalankan aplikasinya dengan cara dipanggil lewat browser "http://localhost/app1"
- selesai.
